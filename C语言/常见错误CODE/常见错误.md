#### 头文件引用

```
在 config 定义函数 double get_voltage_from_config(cJSON* config, double pressure);
在 voltage.c 中 调用 double voltage = get_voltage_from_config(config, pressure);

get_voltage_from_config 中内部打印的是 15.0 V  但是外部是 0V 

原因是 估计是函数实际定义在 config.c 中返回 double（15.0），但在调用处，编译器期望 int，于是将返回的 double 值“截断”或重新解释为 int，导致值变为 0 或垃圾值

```
