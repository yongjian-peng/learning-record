#### 安装 虚拟机   VirtualBox

```
麒麟 V11 默认使用 dnf 作为包管理器（yum 通常是它的软链接）

sudo dnf install kernel-devel kernel-headers make gcc perl

卸载刚才失败的残留
sudo dnf clean all

使用 .run 后缀文件
wget https://download.virtualbox.org/virtualbox/7.0.26/VirtualBox-7.0.26-168464-Linux_amd64.run
安装必要的编译环境
sudo dnf install -y kernel-devel kernel-headers gcc make perl libIDL elfutils-libelf-devel

赋予权限安装
# 赋予执行权限
chmod +x VirtualBox-7.0.26-168464-Linux_amd64.run
# 运行安装程序
sudo ./VirtualBox-7.0.26-168464-Linux_amd64.run
VirtualBox has been installed successfully"，说明核心文件已经成功布署到了你的麒麟系统里

配置内核模块（最重要）
sudo /sbin/vboxconfig
如果显示 vboxdrv.sh: Starting VirtualBox services.：恭喜你，大功告成！

启动
VirtualBox

google-chrome --no-sandbox


```
