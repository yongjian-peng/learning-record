#### 添加样式

```
设置背景颜色 lv_obj_set_style_bg_color()   
设置字体
设置字体颜色
设置位置：居中(lv_obg_set_align())，left right.
添加回调事件

开关
	添加状态，清除状态， 获取状态，禁用修改状态
复选框
	勾选框和文字的间距 设置文本,添加状态，清除状态
下拉列表
	设置选项内容，设置当前所选项，获取选项内容， 设置方向和图标
滚轮部件
	创建滚轮 设置选项间间隔， 设置选项内容，滚动模式，设置当前所选项，设置可见行数，获取选项内容
滑块部件
	创建滑块，设置大小，当前值，范围值，获取当前值，模式设置(正常，从0开始，有两个旋钮)，设置获取左值。
圆弧部件
	设置范围，当前值，设置圆弧角度，设置旋转角度 获取当前值 设置模式，圆弧绘制速率。
线条部件
	创建线条部件， 设置线条坐标点，设置线条样式，设置Y轴反转。
	
图片部件
	创建图片部件，设置图片源，设置图片偏移，图片重新着色。

```

原件

```
按钮
开关 lv_switch_create(); 
复选框 lv_checkbox_create();
进度条
加载器：loading 
列表：
下拉列表：lv_dropdown_create()
滚轮部件: lv_roller_create()
滑块部件： 和进度条很像，就是多了一个手柄的调整范围
圆弧部件：（背景弧，前景弧，手柄）
线条部件 
```
